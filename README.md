# Szalóki Ádám - Weather Forecast

Ez egy időjárás-előrejelző webalkalmazás, amely `Next.js`, `React`, `TypeScript` és `MUI` használatával készült.

## Első indítás

1. Telepítsd a függőségeket:

   ```bash
   npm install
   ```

2. Indítsd el a fejlesztői szervert:

   ```bash
   npm run dev
   ```

3. Nyisd meg az alkalmazást böngészőben:

   ```text
   http://localhost:3000
   ```

## Hasznos parancsok

- `npm run dev` - fejlesztői szerver indítása
- `npm run build` - production build készítése
- `npm run start` - production build futtatása
- `npm run lint` - kódellenőrzés futtatása
